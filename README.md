# 解送人犯法定障礙事由扣除時間計算機

> 適用於警察機關執行解送人犯作業，自動計算法定障礙事由扣除時間，並產出解送記錄表與候詢人入出室通知單。
>
> ---
>
> ## 功能特色
>
> ### 核心計算
> - 輸入逮捕時間與各項法定障礙事由，自動計算扣除後的有效偵查時間
> - - 支援障礙事由時間允許設定為未來時間
>   - - 出生日期萬年歷驗證、24 小時制時間格式驗證
>    
>     - ### 多名嫌犯支援（v3.0+）
>     - - 可同時管理多名嫌犯資料（標籤頁切換）
>       - - 每名嫌犯獨立計算
>        
>         - ### 本地存檔（v3.0+）
>         - - 案件資料儲存於瀏覽器本地（IndexedDB），關閉後可重新載入
>           - - 支援案件讀取 Modal
>            
>             - ### 解送記錄表匯出
>             - - 一鍵產出 Word（.docx）格式解送記錄表
>               - - 記錄表格式比照官方範例（含 ■/□ 勾選符號）
>                 - - 批次匯出：直接儲存至下載資料夾，無需選擇路徑（v3.1.2+）
>                  
>                   - ### 候詢人入出室通知單（v3.0+）
>                   - - 依公文格式還原 A4 直向範本
>                     - - 入室＋出室合併為同一份檔案，檔名含日期、姓名、人數
>                       - - 固定列高、簽章區無內格線、核章列 5cm
>                         - - 通知單標頭格式修正（v3.1.5）
>                          
>                           - ### UI 介面
>                           - - 3D 立體專業介面（v3.1.5）
>                             - - UI 預設值優化（v3.1.4）
>                               - - 離線可用（單一 index.html，無需伺服器）
>                                
>                                 - ### 日出時間查詢
>                                 - - 整合 NOAA 日出計算
>                                   - - 支援離線備援（v3.1.4）
>
