> >> 設計：文一偵查林正賢　　協作AI：Claude（2026/05）　　發布日期：2026年05月

> >>
> >> ## 一、軟體簡介
> >>
> >> 本工具依據《刑事訴訟法》第93條之1規定，協助偵查人員快速計算「解送地檢署」的法定時限（一般案件16小時、通緝犯24小時），並自動扣除法定障礙事由所經過之時間，精確計算最終解送期限。
> >>
> >> **主要特色：**
> >>
> >> - 無需安裝，開啟網頁即可使用，支援電腦、手機、平板。
> >> - - 同案多名嫌犯（最多20人），各自獨立計算期限，分頁切換管理。
> >>   - - 本地自動存檔，關閉後次日可繼續作業，補輸障礙事由再匯出記錄表。
> >>     - - 自動連線中央氣象署，取得所在縣市官方日出日落時間；離線或網路異常時，自動改用 NOAA 演算法推算，不影響使用。
> >>       - - 一鍵產出 附件12（權利告知書）、附件16（告知本人通知書）、附件17（告知親友通知書）。
> >>         - - 一鍵匯出法定障礙事由記錄表（.txt），格式符合書面作業，可直接貼入解送人犯報告書。
> >>           - - 一鍵產出候詢人入/出室通知單（.docx），每份含存根及送候詢室共2聯。
> >>             - - 批次匯出：一次產出全案所有嫌犯的書表（每人附件12、16、17共3份）。
> >>

> >>
> >> ### 步驟一：填寫案件基本資料
> >>
> >> | 欄位 | 說明 |
> >> |------|------|
> >> | 案由 | 輸入犯罪案由，如：詐欺、傷害、竊盜 |
> >> | 承辦人 | 輸入職稱及姓名，如：警員林正賢 |
> >> | 警察機關 | 先選所屬縣市局處（如：臺北市政府警察局） |
> >> | 所屬機關（分局/大隊） | 再選所屬分局或大隊（如：文山第一分局） |
> >> | 單位（派出所/隊） | 輸入派出所或隊名，如：中正派出所 |
> >> | 機關地址 | 選擇分局後自動帶入，亦可手動修改 |
> >>
> >> > ⚠️ **重要：** 必須先選「警察機關」，再選「所屬機關（分局/大隊）」，待機關地址自動帶入後，系統才會自動查詢當地日出日落時間。

> >> > >
> >> > > 點擊「＋ 新增事由」按鈕，選擇障礙類型（刑訴法第93條之1第1項各款）：
> >> > >
> >> > > 1. 因交通障礙或其他不可抗力之事由所生不得已之遲滯
> >> > > 2. 2. 在途解送期間
> >> > >    3. 3. 依第100條之3第1項規定不得為詢問者（夜間停止訊問）
> >> > >       4. 4. 因被告或犯罪嫌疑人身體健康突發之事由，事實上無法訊問者
> >> > >          5. 5. 被告或犯罪嫌疑人表示已選任辯護人，因等候其辯護人到場致未予訊問者
> >> > >             6. 6. 被告或犯罪嫌疑人不通國語須由其通譯傳譯，因等候其通譯到場致未予訊問者
> >> > >                7. 7. 經檢察官命具保之被告，在候保中者
> >> > >                   8. 8. 犯罪嫌疑人經法院提審之期間
> >> > >                      9. 9. 辯護人與偵查中受拘提或逮捕之被告或犯罪嫌疑人接見經過之時間
> >> > >                        
> >> > >                         10. > 💡 **智慧填入：** 選擇第3項（夜間停止訊問）時，系統自動將「結束時間」設定為隔日日出時間，可再手動調整。

> >> > >                             >
> >> > >                             > - **表頭：** 案由、是否為通緝犯、犯罪嫌疑人姓名、承辦人、警察機關、所屬機關、單位、機關地址、拘提/逮捕時間及地點

> >> > >                             >
